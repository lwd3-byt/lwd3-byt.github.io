SpringBoot中的常用模块

pojo 层：将数据库中的表对应成 java 中的 Class

mapper 层(也叫Dao层)：将 pojo 层的 Class 中的操作，映射为 sql 语句

service 层：写具体的业务逻辑，组合使用 mapper 中的操作

controller 层：负责请求转发，接受页面过来的参数，并将其传到 service 层处理；接到返回值，再传给页面
